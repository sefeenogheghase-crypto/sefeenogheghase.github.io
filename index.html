🗄 A — Employee Database SQL Query Set (Week 6)

Wrote 5 progressive SQL queries on a multi-table employee database (employee_demographics, employee_salary, parks_departments) covering INNER JOINs, GROUP BY aggregations, subqueries, LEFT JOINs with COALESCE, and CASE logic for conditional bonus categorisation.

📄 View Full Assignment
Query 1 — INNER JOIN across 3 tables
SELECT
  CONCAT(ed.first_name, ' ', ed.last_name) AS full_name,
  es.occupation, pd.department_name
FROM employee_demographics AS ed
JOIN employee_salary AS es ON ed.employee_id = es.employee_id
JOIN parks_departments AS pd ON es.dept_id = pd.department_id;
Query 2 — GROUP BY Salary Aggregation
SELECT pd.department_name,
  AVG(es.salary) AS average_salary,
  MIN(es.salary) AS minimum_salary,
  MAX(es.salary) AS maximum_salary
FROM employee_salary AS es
JOIN parks_departments AS pd ON es.dept_id = pd.department_id
GROUP BY pd.department_name;
Query 3 — Subquery (Above-Average Salary)
SELECT first_name, last_name, salary
FROM employee_salary
WHERE salary > (SELECT AVG(salary) FROM employee_salary);
Query 4 — LEFT JOIN with COALESCE
SELECT ed.first_name, ed.last_name,
  COALESCE(es.occupation, 'not assigned') AS occupation
FROM employee_demographics AS ed
LEFT JOIN employee_salary AS es ON ed.employee_id = es.employee_id;
Query 5 — CASE Statement (Bonus Categorisation)
SELECT ed.first_name, ed.last_name, pd.department_name,
  CASE
    WHEN pd.department_name = 'parks and recreation' THEN 'eligible for 10% bonus'
    WHEN pd.department_name = 'finance' THEN 'eligible for 15% bonus'
    ELSE 'no bonus'
  END AS bonus_category
FROM employee_demographics AS ed
JOIN employee_salary AS es ON ed.employee_id = es.employee_id
JOIN parks_departments AS pd ON es.dept_id = pd.department_id;
